TOPIC — 01 / 06 🧬 생물

기후변화와 생물

기후변화는 생물의 발아·생장·번식 시기와 서식지, 종 다양성을 바꾼다. 직접 식물을 키우고 학교 주변을 조사하며, 공공데이터와 비교해 기후변화가 생물에 미치는 영향을 탐구해보자.

01
🧪 실험★☆☆ 쉬움

온도 조건이 다르면 씨앗 발아 속도가 달라질까?

+
탐구 방법
같은 씨앗(콩·무순)을 냉장(5°C) / 실온(20°C) / 따뜻한 곳(30°C) 세 조건에서 발아시키고 발아율과 발아 일수를 매일 기록.
준비물
콩 또는 무순 씨앗, 화분 3개, 솜, 온도계, 냉장고, 따뜻한 공간
✦ 핵심 포인트
기후 온난화가 식물 생장 계절을 앞당기는 현상을 실험으로 재현. 대조군·실험군 설계 개념 학습.
02
🧪 실험★★☆ 보통

물풀(엘로데아)의 광합성 속도는 온도에 따라 달라질까?

+
탐구 방법
엘로데아를 차가운 물(10°C) / 실온(20°C) / 따뜻한 물(30°C)에 넣고 빛을 쪼여 1분간 발생하는 기포 수 측정.
준비물
엘로데아(물풀), 비커 3개, 조명, 온도 조절 가능한 물, 초시계
✦ 핵심 포인트
온도와 광합성 속도의 관계를 정량적으로 측정. 기후변화가 탄소 순환에 미치는 영향으로 연결 가능.
03
📏 측정★☆☆ 쉬움

우리 학교에서 장소(아스팔트·잔디·화단·나무 밑)에 따라 사는 생물이 다를까?

+
탐구 방법
교내 4개 구역 설정 → 스마트폰 앱 '네이처링'으로 식물·곤충·새 종 목록 작성 → 구역별 종 수·종류 비교.
준비물
스마트폰, 네이처링 앱, 돋보기, 기록지
✦ 핵심 포인트
서식 환경 다양성과 생물다양성의 관계를 직접 확인. 계절마다 반복하면 시간에 따른 변화도 관찰 가능.
04
📊 데이터★☆☆ 쉬움

우리나라 벚꽃 개화 시기는 최근 30년간 얼마나 앞당겨졌을까?

+
탐구 방법
기상청 생물계절 관측 자료(서울 벚꽃 개화일) 30년치 → 연도별 그래프 → 같은 기간 3월 평균 기온과 비교.
준비물
기상청 기상자료개방포털
✦ 핵심 포인트
개화일과 기온의 상관관계가 명확하게 나타남. '1°C 오르면 며칠 빨라지는가' 계산까지 가능.
TOPIC — 02 / 06 🌾 먹거리

기후변화와 먹거리

기온이 오르면서 우리 밥상이 바뀌고 있다. 직접 식물을 키우며 가뭄·염류화 같은 기후 조건의 영향을 실험하고, 공공데이터로 우리나라 농업 지도의 변화를 추적해보자.

01
🧪 실험★☆☆ 쉬움

물 공급량이 다르면 식물 성장이 어떻게 달라질까? (모의 가뭄 실험)

+
탐구 방법
같은 씨앗(강낭콩)을 심은 화분 3개에 물을 충분히·보통·거의 안 주는 세 조건으로 2주간 키우며 키·잎 색·잎 수를 매일 기록.
준비물
강낭콩 씨앗, 화분 3개, 흙, 물, 자
✦ 핵심 포인트
가뭄이 식물 생장에 미치는 영향을 수치로 확인. 기후변화로 인한 가뭄 빈도 증가와 연결 가능.
02
🧪 실험★☆☆ 쉬움

토양 염분 농도가 높으면 식물이 잘 자라지 못할까? (해수면 상승 토양 염류화 모델)

+
탐구 방법
강낭콩 화분 4개에 소금 농도 다른 물(0% / 0.5% / 1% / 3%)로 2주간 키우며 발아율과 성장 키 기록.
준비물
강낭콩, 화분 4개, 소금, 물, 전자저울, 자
✦ 핵심 포인트
해수면 상승으로 해안 농지가 염류화되는 문제를 직접 재현. 실험 결과가 명확히 나타남.
03
📊 데이터★☆☆ 쉬움

사과 주산지는 최근 30년간 얼마나 북쪽으로 이동했을까?

+
탐구 방법
농촌진흥청 과수 재배 현황 데이터 → 시·도별 사과 재배 면적 1990년대 vs 2020년대 비교 → 지도에 시각화 → 기온 상승 데이터와 비교.
준비물
농촌진흥청 농업통계, 기상청 기온 데이터, 지도
✦ 핵심 포인트
지도 시각화가 매우 인상적. '사과는 이제 경북이 아닌 강원도에서' 결과가 직관적으로 설득력 있음.
04
📊 데이터★☆☆ 쉬움

우리나라 아열대 작물 재배 면적은 기온 상승과 함께 늘어났을까?

+
탐구 방법
농촌진흥청 아열대 작물(망고·바나나·파파야 등) 재배 현황 연도별 데이터 + 기상청 기온 데이터 → 비교 그래프 작성.
준비물
농촌진흥청 농업통계, 기상청 기상자료개방포털
✦ 핵심 포인트
결과가 흥미롭고 학생들이 체감 가능한 소재. 재배 면적 변화가 기온 변화를 따라가는 패턴이 뚜렷함.
TOPIC — 03 / 06 🌊 바다

기후변화와 바다

기후변화로 해수 온도가 오르고 바닷물이 산성으로 변하며 해수면이 높아지고 있다. 우리 손으로 직접 실험하고 데이터를 분석하여 바다에 무슨 일이 일어나고 있는지 밝혀보자.

01
🧪 실험★☆☆ 쉬움

이산화탄소가 많아지면 바닷물이 산성으로 변할까?

+
탐구 방법
탄산음료(높은 CO₂) vs 생수(낮은 CO₂)에 달걀껍데기·조개껍데기를 넣고 무게 변화와 기포 발생을 시간별로 측정한다. BTB 용액으로 산성도 변화도 확인.
준비물
탄산수, 생수, 달걀껍데기, 조개껍데기, 전자저울, BTB 용액, 비커
✦ 핵심 포인트
CO₂ 농도 차이가 탄산칼슘 용해 속도에 미치는 영향을 수치로 확인. 해양 산성화의 핵심 메커니즘을 직접 체험.
02
🧪 실험★☆☆ 쉬움

육지 빙하가 녹을 때와 바다 빙하(해빙)가 녹을 때 해수면 변화는 다를까?

+
탐구 방법
수조 두 개 준비 — ①빙하 얼음을 수조 밖 받침 위에 올린 경우 ②얼음을 수조 안 물 위에 띄운 경우 → 녹은 후 수위 변화 비교 측정.
준비물
수조 2개, 얼음, 눈금자, 물
✦ 핵심 포인트
아르키메데스 원리를 체험하며 '왜 바다 빙하보다 육지 빙하가 더 위험한가'를 직접 데이터로 증명.
03
📊 데이터★☆☆ 쉬움

우리나라 해수면은 최근 30년간 얼마나 높아졌을까?

+
탐구 방법
국립해양조사원 연안 해수면 관측 데이터(목포·제주·속초 등 3개 지점) 다운로드 → 연도별 평균 해수면 높이 그래프 → 상승 속도(mm/년) 계산.
준비물
국립해양조사원 공공데이터, 모눈종이 or 엑셀
✦ 핵심 포인트
지점별 상승 속도 차이도 비교 가능. 실험 결과(빙하 녹기)와 연결하면 탐구 완성도 높아짐.
04
📊 데이터★☆☆ 쉬움

바닷물 온도는 계절마다 얼마나 다를까? 과거보다 높아졌을까?

+
탐구 방법
국립해양조사원 해수 표층 수온 월별 데이터 → 같은 달 10년 단위 평균 비교 → 계절별 변화 그래프 작성.
준비물
국립해양조사원 공공데이터
✦ 핵심 포인트
수온 상승이 해양 생태계에 미치는 영향(산호 백화 등)과 연결하면 스토리 완성.
TOPIC — 04 / 06 ❤️ 건강

기후변화와 건강

기온이 오를수록 사람의 몸에도 직접적인 위협이 된다. 체감온도 실험과 온열질환 통계 분석을 통해 기후변화가 건강에 미치는 영향을 탐구해보자.

01
🧪 실험★☆☆ 쉬움

같은 온도라도 습도가 높으면 더 덥게 느껴질까? (체감온도 실험)

+
탐구 방법
같은 공간에서 습도만 달리하여(가습기 사용) 체감 온도와 실제 온도를 기록. 습도별 체감온도 공식으로 계산값과 비교.
준비물
온습도계, 가습기, 기록지
✦ 핵심 포인트
폭염의 위험이 단순히 기온만이 아님을 수치로 확인. 온열질환 데이터와 연결하면 탐구 완성도 높아짐.
02
🧪 실험★☆☆ 쉬움

자외선 차단제 농도가 다르면 자외선 차단 효과도 다를까?

+
탐구 방법
UV 감지 비드(자외선에 색 변하는 구슬)에 자외선 차단제를 농도별로(무도포·SPF15·SPF50) 바르고 햇빛에 노출 후 색 변화 관찰.
준비물
UV 감지 비드, 자외선 차단제(여러 SPF), 햇빛 or 자외선 램프
✦ 핵심 포인트
기후변화로 자외선 노출이 증가하는 맥락과 연결. 실험 결과가 시각적으로 뚜렷해 흥미도 높음.
03
📊 데이터★☆☆ 쉬움

최근 10년간 폭염 일수와 온열질환자 수는 함께 증가했을까?

+
탐구 방법
기상청 폭염 일수(연도별) + 질병관리청 온열질환 감시체계 통계 → 두 변수 꺾은선 그래프 비교 → 상관관계 분석.
준비물
기상청 기상자료개방포털, 질병관리청 온열질환 통계
✦ 핵심 포인트
두 공공기관 데이터 연결이 핵심. 상관관계와 인과관계의 차이를 배우는 좋은 사례.
04
📊 데이터★★☆ 보통

폭염이 가장 심한 지역과 덜한 지역의 온열질환자 수는 차이가 날까?

+
탐구 방법
기상청 지역별 폭염 일수 + 질병관리청 지역별 온열질환 통계 → 지역별 비교 막대그래프 → 인구 10만 명당 환자 수로 보정해 공정 비교.
준비물
기상청 기상자료개방포털, 질병관리청 통계
✦ 핵심 포인트
'인구 보정' 개념을 스스로 도입하면 우수 탐구. 도농 격차, 냉방 인프라 차이와 연결 가능.
TOPIC — 05 / 06 ⚡ 에너지

기후변화와 에너지

기후변화를 막으려면 에너지를 바꿔야 한다. 태양광·단열 실험을 통해 신재생에너지의 원리를 탐구하고, 실제 에너지 전환 데이터로 우리나라의 변화를 분석해보자.

01
🧪 실험★★☆ 보통

태양광 패널의 각도를 바꾸면 발전량이 달라질까?

+
탐구 방법
소형 태양광 패널(5V)을 0°/30°/45°/60°/90°로 기울여 멀티미터로 전압·전류 측정 → 발전량(P=VI) 계산 → 최적 각도 탐구.
준비물
소형 태양광 패널, 멀티미터, 각도기, 조명 or 햇빛
✦ 핵심 포인트
우리 지역 위도(약 37°)와 최적 각도 비교. 실제 태양광 설치 각도와 연결하면 생활 과학으로 확장.
02
🧪 실험★☆☆ 쉬움

단열재 종류에 따라 온도 유지 능력이 다를까?

+
탐구 방법
같은 크기 상자에 스티로폼·신문지·면·비닐 등 단열재를 각각 넣고 뜨거운 물(50°C)을 담은 컵을 올린 후 10분마다 온도 측정.
준비물
상자 4개, 스티로폼, 신문지, 면, 비닐, 온도계, 뜨거운 물
✦ 핵심 포인트
단열 성능 차이를 수치로 비교. 건물 단열재와 에너지 절약의 연결로 탐구 확장 가능.
03
🧪 실험★☆☆ 쉬움

검은 캔과 흰 캔, 태양열을 흡수하는 속도는 얼마나 차이 날까?

+
탐구 방법
검은색으로 칠한 캔과 흰색 캔에 같은 양의 물을 넣고 햇빛(or 램프) 아래 놓은 후 5분마다 온도 측정.
준비물
빈 캔 2개, 검은색·흰색 페인트 or 종이, 온도계, 물
✦ 핵심 포인트
태양열 에너지 흡수 원리를 직접 측정. 태양열 온수기·태양광 패널 색깔 설계와 연결 가능.

TOPIC — 06 / 06 🌪️ 재난

기후변화와 재난

기후변화는 집중호우·홍수·산사태·폭염 같은 재난을 더 자주, 더 강하게 만든다. 직접 모형 실험으로 재난의 원리를 재현하고, 공공데이터로 우리나라 재난의 변화를 추적해보자.

01
🧪 실험★☆☆ 쉬움

표면 종류(잔디·흙·아스팔트·자갈)에 따라 물이 스며드는 속도가 다를까? (도시 홍수 모델)

+
탐구 방법
화분 4개에 각각 잔디·흙·아스팔트 모형(타일)·자갈을 넣고 같은 양의 물을 부어 밑으로 빠져나오는 시간과 양 측정. 안 빠진 물(=지표에 고이는 물)도 비교.
준비물
화분 4개, 잔디·흙·자갈·타일, 물, 비커, 초시계
✦ 핵심 포인트
도시화로 불투수면이 늘면서 도시 홍수 위험이 높아지는 이유를 직접 확인. 그린 인프라의 필요성으로 연결 가능.
02
🧪 실험★★☆ 보통

경사면에 식물이 있으면 토사 유출(산사태)을 얼마나 막아줄까?

+
탐구 방법
쟁반에 같은 경사로 흙을 쌓아 ①맨흙 ②잔디·콩나물 뿌리가 자란 흙 두 모형을 만들고, 같은 양의 물을 같은 높이에서 부어 흘러내린 흙(토사) 무게를 비교 측정.
준비물
쟁반 2개, 흙, 잔디 or 콩나물, 물뿌리개, 받침대, 전자저울
✦ 핵심 포인트
집중호우 시 산사태·토양 침식의 원리를 모형으로 재현. 식생 복원이 재난을 줄이는 자연 기반 해법임을 체험.
03
📊 데이터★☆☆ 쉬움

우리 지역 집중호우(시간당 30mm 이상) 빈도는 최근 30년간 늘어났을까?

+
탐구 방법
기상청 기상자료개방포털 → 우리 지역 측정소 시간당 강수량 30년치 → 10년 단위로 집중호우 발생 횟수 비교 → 총 강수량 변화와도 비교.
준비물
기상청 기상자료개방포털
✦ 핵심 포인트
총 강수량보다 집중호우 빈도 변화가 더 극적인 경우가 많음. '양은 비슷한데 왜 홍수가 자주 날까?'로 심화 탐구 연결.
04
📊 데이터★★☆ 보통

우리나라 자연재난(호우·태풍·폭염) 피해는 최근 어떻게 변하고 있을까?

+
탐구 방법
행정안전부 재해연보(국가재난정보) → 연도별 자연재난 유형별 피해액·피해 건수 → 호우·태풍·폭염 추이 그래프 작성 → 기온·강수 데이터와 비교.
준비물
행정안전부 재해연보, 기상청 기상자료개방포털
✦ 핵심 포인트
재난 피해를 '돈'과 '횟수'로 체감. 물가 상승 효과를 함께 고려하면 비판적 데이터 해석 훈련 가능.
